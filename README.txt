Controls:
W go down
S go up
D go left
A go right
G store your current position
R return to that stored position in a teleport manner
F throw the teleportation device, you teleport to wherever it lands
M restart playthrough
TAB open the options men√∫
SPACE jump
SHIFT walk faster
2 Sets the ball speed to slow
3 Sets the ball speed to fast
4 Sets it back to default
H used to display the tips
